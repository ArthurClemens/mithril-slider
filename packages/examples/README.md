## Viewing locally

* `git clone https://github.com/ArthurClemens/mithril-slider.git`
* `cd mithril-slider`
* Install Lerna globally: `npm install -g lerna`
* `npm install`
* `lerna init`
* `lerna bootstrap`
* `cd packages/examples`
* `npm run watch`
* Open browser at [localhost:8080](http://localhost:8080/)
